ระบบบริหารยานพาหนะ(ระบบจองรถ)

พัฒนาด้วยภาษา Yii2 Framework เป็นภาษา php ฐานข้อมูล MariaDB สามารถใช้งานผ่าน Docker compose ได้

กลับสู่หน้าหลัก
หน้าการจองรถ

เลือกวันที่จะจองจากนั้นจะแสดงรายการรถที่ว่าง

หน้าบันทึกจองรถ

กรอกรายละเอียดในการจอง

แบบฟอร์มเบิกค่าเดินทาง

ระบบแบบฟอร์มเบิกค่าเดินทางกรอกข้อมูลเพื่อพิมพ์ใบเบิกค่าเดินทาง

แสดงรายละเอียดการขอใช้ยานพาหนะ

หน้าแสดงรายละเอียดเกี่ยวกับรายละเอียดการจอง สามารถ แก้ไข ยกเลิก หรือพิมพ์ใบเบิกค่าเดินทางได้

ระบบโปรไฟล์ส่วนตัว

สามารถเปลี่ยนรหัสผ่านและข้อมูลอื่นๆได้

ระบบ Authentication

ระบบยืนยันตัวตนโดยกรอก username password หลังจากนั้นจะเข้าสู่ระบบตามสิทธิการเข้าใช้งานตามที่ถูกกำหนดค่าเอาไว้ในแต่ละส่วน

ระบบ line official account (สำหรับพนักงานขับรถ)

เมนูหน้าหลัก สามารถดูการแจ้งเตือนการจองเข้ามา

ระบบแสดงรายการที่จองเข้ามา

แสดงรายการจองยานพาหนะและสารมารถเลือกรับภาระกิจได้

ระบบแสดงรายการที่จองเข้ามา

เมื่อเลืกรับภาระกิจของพนักงานขับรถแล้วจะมีการแจ้งเตือนเพื่อยืนยันอีกครั้ง
